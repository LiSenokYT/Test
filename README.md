# Архив Брони

Современная энциклопедия военной техники с детальной информацией о различных образцах вооружения.

## Структура проекта

- `index.html` - Главная страница
- `styles/main.css` - Основные стили
- `js/script.js` - JavaScript функциональность
- `assets/images/` - Изображения и медиафайлы

## Установка и запуск

1. Клонируйте репозиторий
2. Откройте `index.html` в браузере
3. Для разработки можно использовать Live Server

## Технологии

- HTML5
- CSS3 (Grid, Flexbox, Custom Properties)
- Vanilla JavaScript
- Responsive Design
